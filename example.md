# Notepad--MarkdownUDF
An user defined file for Markdown

This UDF supports:

# This is title

+ list1
+ list2

* list1
* list 2

- list1
- list2

Special font *italics font*, **bold font**, _italics font_, __bold font__ and normal font.

~~deleted line~~, and ~~not~~ deleted. 

>quote1
>quote2

`hello world`

```
char*a="hello world";
printf("%s",a);
```
